## Description

[Postman Collection](https://documenter.getpostman.com/view/25716495/2sA2r834tP) to test endpoints.

## Installation

```bash
$ yarn install
```

## Database Setup

```bash
$ docker compose up -d dev-db
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash


# e2e tests
$ yarn run test:e2e


```
